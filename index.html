Javascript Quiz Game

Instructions

The following is a Javascript Knowledge Trivia Game. The overall purpose of this web app is to make learning JavaScript fun. To begin playing, click on the Start Game button. There you will be presented with a timer and a set of questions that you must answer. For every question that you get right you will be rewarded with 3 points to your score. However, if you answer incorrectly you will be penalized by having 5 seconds subtracted from your overall time. Once you finish, enter your name to the High Score board.
To get a better experience open the developer console to view the web apps inner working and messages.
